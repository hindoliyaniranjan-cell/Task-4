Phishing Email Analysis Report

Sample Source: Suspicious email claiming to be from “PayPal Support”

1. Sender Address

From: support@paypall-security.com

Legitimate domain is paypal.com.

Indicator: Spoofed domain (extra 'l').

2. Header Analysis

SPF check failed → sending IP 185.32.55.19 not authorized for paypal.com.

DKIM signature missing.

Received from shows relay in Eastern Europe, not PayPal servers.

Indicator: Header anomalies and authentication failures.

3. Links

Visible link: https://paypal.com/login

Actual link (hovered): http://secure-update.ru/paypal

Indicator: Mismatched URL, domain hosted in Russia.

4. Attachments

File: Invoice_Update.zip containing Invoice_Update.exe.

Indicator: Executable attachment disguised as document.

5. Language & Tone

Subject: URGENT: Verify Your Account Now!

Body uses threatening language ('Your account will be locked in 24 hours').

Indicator: Psychological pressure tactic.

6. Spelling/Grammar

Example: 'Your accont will be terminate if not verify.'

Indicator: Poor grammar, unprofessional wording.
